Placez ici le logo Xeilom au format WebP :

- Fichier attendu : `logo.webp`
- Chemin public : `public/brand/logo.webp` (dans votre projet hôte)
- Dimensions recommandées : ~280×68 px (ratio conservé en header et PDF)

Le kit expose l'URL via `BRAND_LOGO_URL` et charge le logo pour les PDF via `loadBrandLogoForPdf()`.
